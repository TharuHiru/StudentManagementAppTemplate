# Student Management App (Android Java)

## Overview
This is a **simple Student Management Android application** created using **Android Studio and Java**.  
The app is designed as a **basic template** for beginners to understand CRUD operations in Android.

It allows users to:
- Add students  
- Edit student details  
- Delete students  
- View student records  

The app uses an **in-memory database (SQLite)** for simplicity and learning purposes.

---

## Technologies Used
- **Android Studio**
- **Java**
- **SQLite (In-Memory Database)**
- **XML (UI Design)**

---

## Features
- Simple and beginner-friendly UI
- Add student details (e.g., name, ID, etc.)
- Edit existing student records
- Delete student records
- Local data storage using SQLite
- Lightweight and easy-to-understand code structure

## How to Run the Project
1. Clone the repository:
   ```bash
   git clone https://github.com/TharuHiru/StudentManagementAppTemplate.git
2. Open Android Studio
3. Click Open an Existing Project
4. Select the project folder
5. Let Gradle sync
6. Run the app on an emulator or physical device
